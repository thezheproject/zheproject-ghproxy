这计划 GitHub 文件加速

GitHub 文件链接带不带协议头都可以，支持 Releases、Archives 以及文件，右键复制出来的链接都是符合标准的，更多用法、克隆加速请参考这篇文章。

Releases 和 Archives 使用 Cloudflare 加速，文件会转至 JsDelivr 的 Fastly 服务器。

注意： 不支持项目文件夹，只能加速这计划持有的仓库。加速其它非计划项目，请在这里寻找合适的节点。
